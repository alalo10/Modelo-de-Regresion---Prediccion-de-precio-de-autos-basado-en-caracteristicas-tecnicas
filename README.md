## Planteamiento del problema
En el mercado automovilístico, el precio de un vehículo suele verse influenciado por factores externos como la marca, la reputación del fabricante, etc. Sin embargo, también existen características técnicas propias del automóvil que pueden incidir de manera significativa en su valor comercial.

Ante esta situación, surge la necesidad de analizar en qué medida dichas características permiten estimar su precio. Comprender esta relación puede resultar útil para identificar cuáles atributos influyen más en la valoración de los vehículos y para construir modelos que permitan realizar predicciones basadas únicamente en información técnica en un mercado estadounidense.

Por lo tanto, en este proyecto se propone desarrollar un modelo de regresión que permita predecir el precio de los automóviles a partir de sus características técnicas, con el fin de analizar la relación entre dichas variables y el valor del vehículo en el mercado.

## Objetivo

Desarrollar un modelo de regresión que permita predecir el precio de automóviles
a partir de sus características técnicas, con el fin de analizar la relación entre
dichas variables y el valor del vehículo en el mercado estadounidense.



## Dataset

| Atributo       | Detalle                              |
|----------------|--------------------------------------|
| **Registros**  | 305 filas                            |
| **Columnas**   | 12 (7 categóricas, 5 numéricas)      |
| **Tamaño**     | ~28.7 KB                             |
| **Valores nulos** | Ninguno                           |
| **Variable objetivo** | `price` (precio en USD)       |

### Variables del dataset

| Variable         | Descripción                                          |
|------------------|------------------------------------------------------|
| `CarName`        | Nombre del modelo del automóvil                      |
| `fueltype`       | Tipo de combustible (gasolina, diésel)               |
| `aspiration`     | Tipo de aspiración del motor (estándar / turbo)      |
| `doornumber`     | Cantidad de puertas                                  |
| `carbody`        | Tipo de carrocería (sedán, coupé, SUV, etc.)         |
| `enginetype`     | Configuración del motor                              |
| `cylindernumber` | Número de cilindros                                  |
| `enginesize`     | Tamaño del motor en cc                               |
| `boreratio`      | Relación del diámetro del cilindro                   |
| `horsepower`     | Caballos de fuerza                                   |
| `peakrpm`        | RPM máximas del motor                                |
| `price`          | Precio del automóvil en USD *(Ground Truth)*         |
